jQuery

jQuery is an open source JavaScript library that simplifies the interactions
between an HTML document, or more precisely the Document Object Model (DOM) and JavaScript. Query is free, open source software

Historique

jQuery was first released in January 2006 by John Resig at BarCamp NYC. It is currently headed by Timmy Wilson and maintained by a team of developers. Nowadays, jQuery is widely used technology. Most of the websites are using jQuery

Features of jQuery

It is written only in javascript. It is external - written by a third party - and its size is a little 196 kilobytes. Easy selection of elements in HTML page.

jQuery Events

Query is one of the many library features that is created and used for javascript codes , in order to expand the functional boundaries of javascript. The jQuery Events can be defined as any activity or even a click that is made on the web page created using javascript and jQueries.

jQuery Effects

jQuery is one of the most popular javascript libraries which is designed to simplify the HTML DOM i.e. the document object model properties such as manipulation and tree traversal. Other properties such as event handling, Ajax and CSS animation is also simplified

jQuery DOM Manipulation

One very important part of jQuery is the possibility to manipulate the DOM. jQuery comes with a bunch of DOM related methods that make it easy to access and manipulate elements and
DOM = Document Object Model

jQuery Selectors

Query selectors allow you to select and manipulate HTML element(s). jQuery selectors are used to "find" (or select) HTML elements based on their name, id, classes, types, attributes, values of attributes and much more.
